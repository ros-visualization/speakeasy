waitForRosNode(nodeName, timeout=None, waitMessage=None, provideTimeInfo=None)
Static Method

source code 

Wait for a named ROS node to start up. Wait can just hang, or a given message may be provided that is printed every second. Additionally, a count-down in seconds may be added automatically to the given message, if a timeout is provided. Or, if no timeout is provided, a count-up maybe provided to keep track of how long the (indefinite) wait has been going on.

Parameters:
  • nodeName (string) - entire, or partial name of ROS node as shown in 'rosnode list'
  • timeout (int) - Number of seconds after which a ValueError is to be raised.
  • waitMessage (string) - any message to be printed every second during the wait.
  • provideTimeInfo (boolean.) - provide time info in the 1-second status messages. If a timeout is provided, the time info will be a count-down, else a count-up. If a waitMessage is provided, the timing info is appended to that message.
Raises:
  • NotImplementedError - if service is not running before timeout elapses.
